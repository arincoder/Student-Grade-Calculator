# Student Grade Calculator 🎓

A Python-based Student Grade Calculator that collects student scores, calculates academic performance, and generates a detailed report.

This project is part of my Python learning journey and represents my progression from beginner programs into more structured applications. It focuses on improving my understanding of functions, lists, loops, input validation, and clean program organization.

---

## Features

- Collects student name and number of subjects
- Accepts multiple subject scores
- Validates scores to ensure they are between 0 and 100
- Calculates:
  - Total score
  - Average score
  - Highest score
  - Lowest score
  - Letter grade
  - Pass/Fail status
- Allows users to calculate grades for multiple students
